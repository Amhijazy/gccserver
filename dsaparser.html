DSA Local Parser

Installable Version

The DSA Local parser installable version allows you to install parser on your Windows operating system once, then you can right-click any DSA file and select Send To > Local DSA Parser to parse it.

Installation Steps:
  • First, Download the dsa-local.exe installer
  • Once it is downloaded, double click the installer to start the installation
  • When the installation is successful, it will show you the tools version number
  • If the installation didn't go through for any reason, contact bcford@us.ibm.com
Usage:
  • Right-click the DSA file that you need to parse and select Send To > Local DSA Parser
  • You will be presented by a number of options, select the one you need

Batch-Like Portable Version

This local parser version is portable and doesn't need installation. It allows you to choose where to save the parsed DSA html files as well as which DSA version is used for parsing.

Installation Steps:
  • Download the DSA local parse v2.zip
  • Unzip the downloaded folder
Usage:
  • Open the extracted folder
  • Double-click the DSA local parser v2.exe
  • Follow the batch file instructions
